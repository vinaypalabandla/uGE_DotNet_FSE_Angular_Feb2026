DOM Manipulations

Add/Remove  elements at runtime:  



var paraObj = document.createElement("p");
paraObj.setAttribute("id", "p1");
paraObj.innerHTML =  "Adding new DOM elements at runtime";

document.body.appendChild(paraObj);	//  append to the body 
document.body.removeChild(paraObj);	//  append to the body 

var  divObj = document.geteElementById("div1");
divObj.appendChild(paraObj);   // append to the div tag 
divObj.removeChild(paraObj);   // append to the div tag

	var, let and const
	~~~~~~~~~~~~~~~~
	
	var ---  entire function scope, accessible before declaration. 
	let  ---  block scope, not accessible before declaration (error)
	const  ---  block scope, not accessible before declaration (error), not re-assignable
